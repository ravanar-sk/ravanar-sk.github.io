🎮 About the App

Bingo is an offline multiplayer game designed for friends and family. No internet connection is required to play.

❓ Common Help

  • Restart the app if a game gets stuck
  • Ensure devices are on the same network for multiplayer
  • Update to the latest version for best experience

🔒 Privacy
