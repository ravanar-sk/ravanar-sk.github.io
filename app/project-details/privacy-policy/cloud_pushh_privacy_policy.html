Privacy Policy – Cloud Pushh

Privacy Policy

Cloud Pushh

Effective Date: March 27, 2026

Overview

Cloud Pushh is a macOS desktop application used to send test push notifications to mobile applications. It is designed for developers and testers to validate push notification workflows.

Information You Provide

  • Push notification payloads and message content
  • Device tokens, bundle IDs, topics
  • APNs / FCM credentials and configuration

Network Communication

Cloud Pushh communicates only with official public endpoints:

  • Apple Push Notification service (APNs)
  • Firebase Cloud Messaging (FCM)

No private servers, proxies, or backend services are used. All requests are sent directly from your device.

Local Storage

Some data such as recent payloads or configurations may be stored locally on your Mac to improve usability.

Data Usage

Data is used only to send test notifications. We do not sell, track, or analyze your data.

Security

You are responsible for protecting credentials used in the app.
